DMPlexCreateCoarsePointIS

Synopsis

#include "petscdmplex.h"   
PetscErrorCode DMPlexCreateCoarsePointIS(DM dm, IS *fpointIS)

Input Parameter

dm - The coarse DM

Output Parameter

fpointIS - The IS of all the fine points which exist in the original coarse mesh
