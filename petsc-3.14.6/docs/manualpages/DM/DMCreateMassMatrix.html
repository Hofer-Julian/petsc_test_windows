DMCreateMassMatrix

 Gets mass matrix between two DM objects, M_ij = \int \phi_i \psi_j

Synopsis

#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMCreateMassMatrix(DM dac, DM daf, Mat *mat)
Collective on dac

Input Parameter

dac - the DM object
daf - the second, finer DM object

Output Parameter

mat - the interpolation
