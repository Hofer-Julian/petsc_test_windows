VecGetLocalSize

 Returns the number of elements of the vector stored in local memory.

Synopsis

#include "petscvec.h"   
PetscErrorCode  VecGetLocalSize(Vec x,PetscInt *size)
Not Collective

Input Parameter

x - the vector

Output Parameter

size - the length of the local piece of the vector
