VecBindToCPU

 marks a vector to temporarily stay on the CPU and perform computations on the CPU

Synopsis

#include "petscvec.h"   
PetscErrorCode VecBindToCPU(Vec v,PetscBool flg)

Input Parameters

v - the vector
flg - bind to the CPU if value of PETSC_TRUE
