VecScatterGetMerged

 Returns true if the scatter is completed in the VecScatterBegin() and the VecScatterEnd() does nothing

Synopsis

#include "petscvec.h"    
PetscErrorCode  VecScatterGetMerged(VecScatter ctx,PetscBool  *flg)
Not Collective

Input Parameter

ctx - scatter context created with VecScatterCreate()

Output Parameter

flg - PETSC_TRUE if the VecScatterBegin/End() are all done during the VecScatterBegin()

See Also

VecScatterCreate(), VecScatterEnd(), VecScatterBegin()

developer
