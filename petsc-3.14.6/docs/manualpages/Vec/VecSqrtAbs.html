VecSqrtAbs

 Replaces each component of a vector by the square root of its magnitude.

Synopsis

#include "petscvec.h" 
PetscErrorCode  VecSqrtAbs(Vec v)
Not collective

Input Parameter

v - The vector

Output Parameter

v - The vector square root

Note: The actual function is sqrt(|x_i|)
