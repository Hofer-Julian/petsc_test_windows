PetscRegisterFinalize

Synopsis

PetscErrorCode  PetscRegisterFinalize(PetscErrorCode (*f)(void))
Not Collective

Input Parameter

PetscErrorCode (*fun)(void) -

Notes

This is used by, for example, DMInitializePackage() to have DMFinalizePackage() called
