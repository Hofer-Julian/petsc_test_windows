PETSC_DETERMINE

 standard way of passing in integer or floating point parameter where you wish PETSc to compute the required value.

Developer Note

I would like to use const PetscInt PETSC_DETERMINE = PETSC_DECIDE; but for some reason this is not allowed by the standard even though PETSC_DECIDE is a constant value.
