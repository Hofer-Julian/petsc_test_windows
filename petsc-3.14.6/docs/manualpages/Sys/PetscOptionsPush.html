PetscOptionsPush

 Push a new PetscOptions object as the default provider of options Allows using different parts of a code to use different options databases

Synopsis

#include "petscsys.h"   
PetscErrorCode PetscOptionsPush(PetscOptions opt)
Logically Collective

Input Parameter

opt - the options obtained with PetscOptionsCreate()

Notes

Use PetscOptionsPop() to return to the previous default options database

The collectivity of this routine is complex; only the MPI processes that call this routine will have the affect of these options. If some processes that create objects call this routine and others do not the code may fail in complicated ways because the same parallel solvers may incorrectly use different options on different ranks.

See Also
