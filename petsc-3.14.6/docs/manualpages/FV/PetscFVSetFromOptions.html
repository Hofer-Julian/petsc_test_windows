PetscFVSetFromOptions

 sets parameters in a PetscFV from the options database

Synopsis

PetscErrorCode PetscFVSetFromOptions(PetscFV fvm)
Collective on fvm

Input Parameter

fvm - the PetscFV object to set options for

Options Database Key
