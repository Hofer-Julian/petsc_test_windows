PetscFVCreate

 Creates an empty PetscFV object. The type can then be set with PetscFVSetType().

Synopsis

#include "petscfv.h" 
PetscErrorCode PetscFVCreate(MPI_Comm comm, PetscFV *fvm)
Collective

Input Parameter

comm - The communicator for the PetscFV object

Output Parameter

fvm - The PetscFV object
