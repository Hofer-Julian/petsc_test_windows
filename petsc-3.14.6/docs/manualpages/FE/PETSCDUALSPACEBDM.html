PETSCDUALSPACEBDM

 "bdm" - A PetscDualSpace object that encapsulates a dual space for Brezzi-Douglas-Marini elements

Note: This type is a constructor alias of PETSCDUALSPACELAGRANGE. During PetscDualSpaceSetUp(), the correct value of PetscDualSpaceSetFormDegree() is set for H-div conforming spaces. The type of the dual space is then changed to to PETSCDUALSPACELAGRANGE.
