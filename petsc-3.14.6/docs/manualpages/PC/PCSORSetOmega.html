PCSORSetOmega

 Sets the SOR relaxation coefficient, omega (where omega = 1.0 by default).

Synopsis

#include "petscpc.h" 
PetscErrorCode  PCSORSetOmega(PC pc,PetscReal omega)
Logically Collective on PC

Input Parameters

pc - the preconditioner context
omega - relaxation coefficient (0 < omega < 2).

Options Database Key

-pc_sor_omega <omega> - Sets omega

Note

If omega != 1, you will need to set the MAT_USE_INODES option to PETSC_FALSE on the matrix.

See Also

PCSORSetSymmetric(), PCSORSetIterations(), PCEisenstatSetOmega(), MatSetOption()

Level

intermediate
