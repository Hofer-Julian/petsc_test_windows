MatBindToCPU

 marks a matrix to temporarily stay on the CPU and perform computations on the CPU

Synopsis

#include "petscmat.h"  
PetscErrorCode MatBindToCPU(Mat A,PetscBool flg)

Input Parameters

A - the matrix
flg - bind to the CPU if value of PETSC_TRUE
