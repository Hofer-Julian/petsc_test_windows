MatNullSpaceDestroy

 Destroys a data structure used to project vectors out of null spaces.

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatNullSpaceDestroy(MatNullSpace *sp)
Collective on MatNullSpace

Input Parameter

sp - the null space context to be destroyed
