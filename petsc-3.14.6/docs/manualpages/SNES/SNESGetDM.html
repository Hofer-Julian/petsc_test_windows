SNESGetDM

 Gets the DM that may be used by some preconditioners

Synopsis

#include "petscsnes.h"  
PetscErrorCode  SNESGetDM(SNES snes,DM *dm)
Not Collective but DM obtained is parallel on SNES

Input Parameter

snes - the preconditioner context

Output Parameter

dm - the dm
