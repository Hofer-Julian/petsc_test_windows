SNESGetSolution

 Returns the vector where the approximate solution is stored. This is the fine grid solution when using SNESSetGridSequence().

Synopsis

#include "petscsnes.h"  
PetscErrorCode  SNESGetSolution(SNES snes,Vec *x)
Not Collective, but Vec is parallel if SNES is parallel

Input Parameter

snes - the SNES context

Output Parameter

x - the solution

See Also

SNESGetSolutionUpdate(), SNESGetFunction()

Level

intermediate
