PetscDSCreate

 Creates an empty PetscDS object. The type can then be set with PetscDSSetType().

Synopsis

#include "petscds.h" 
PetscErrorCode PetscDSCreate(MPI_Comm comm, PetscDS *prob)
Collective

Input Parameter

comm - The communicator for the PetscDS object

Output Parameter

prob - The PetscDS object
