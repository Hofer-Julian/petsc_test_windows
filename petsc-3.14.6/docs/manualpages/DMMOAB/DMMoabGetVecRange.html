DMMoabGetVecRange

Synopsis

PetscErrorCode DMMoabGetVecRange(Vec vec, moab::Range *range)

Input Parameter

vec - Vec being queried

Output Parameter

range - Entities associated with this Vec. NULL if vec is a native PETSc Vec.
