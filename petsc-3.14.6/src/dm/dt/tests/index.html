DM Examples

DM objects are used to manage communication between the algebraic structures in PETSc (Vec and Mat) and mesh data structures in PDE-based (or other) simulations. See, for example, DMDACreate().

ex1.c: Tests 1D discretization tools
ex2.c: Tests 1D cell-based discretization tools
ex3.c: Tests quadrature
ex4.c: Tests dual space symmetry
ex5.c: Tests affine subspaces
ex6.c: Tests 1D Gauss-Lobatto-Legendre discretization on [-1, 1]
ex7.c: Test the PetscDTAltV interface for k-forms (alternating k-linear maps)
